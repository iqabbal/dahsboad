# navimar-Reporting4Elman

## Getting Started

```sh

git clone https://gitlab.navimarsystems.com:8929/navimar-integration/navimar-reporting4elman.git

cd  navimar-reporting4elman

git checkout backend

sudo docker-compose up 
 
```
